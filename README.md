## 项目简介

本项目是一个面向日常问答与内容辅助生成场景的 Web 端 AI 对话助手。  
在不依赖 Vue、React 等前端框架的前提下，使用原生 JavaScript 完成页面搭建、组件交互、状态管理与接口调用，重点实现了类似 AI 聊天产品中的核心体验，包括：

- 欢迎页快捷提问入口
- AI 对话消息流式返回
- Markdown 内容渲染
- 代码块语法高亮与复制
- 深色 / 浅色主题切换
- 图片上传与本地预览
- 本地存储 API Key 与用户偏好

该项目适合作为前端基础能力与 AI 应用接入能力的展示型项目。

---

## 项目亮点

### 1. 原生 JavaScript 实现完整 AI 对话体验
未使用前端框架，基于原生 HTML、CSS、JavaScript 独立完成页面结构组织、交互逻辑编排与状态更新，体现了对前端基础能力的掌握。

### 2. 接入阿里云百炼大模型 API
通过调用阿里云百炼模型，实现真实的 AI 问答能力，并结合前端流式渲染机制提升交互体验。

### 3. 流式输出 + 打字机效果
AI 回复采用流式展示，内容逐字输出，增强了对话过程的实时感与产品体验。

### 4. Markdown 渲染与代码高亮
对 AI 返回内容进行 Markdown 解析，支持标题、列表、引用、表格、代码块等常见格式，并对代码块增加语法高亮与一键复制功能。

### 5. 更贴近真实产品的交互设计
支持深浅主题切换、回车发送 / Shift + Enter 换行、停止生成、清空会话、图片上传预览等，尽量还原成熟 AI 产品中的常见交互。

## 功能特性

### 首页展示
- 欢迎标题与副标题展示
- 标题渐变色视觉效果
- 4 张快捷建议卡片横向排列
- 点击卡片可直接发起对话

### AI 对话功能
- 接入阿里云百炼模型，实现真实 AI 问答
- 用户消息右侧展示，AI 消息左侧展示并带头像
- 支持流式响应
- 支持打字机逐字显示效果
- 进入对话后自动隐藏欢迎区与快捷卡片

### Markdown 渲染
- 支持标题、列表、引用、表格、代码块等渲染
- 提升 AI 回复内容的可读性与结构化展示效果

### 主题切换
- 支持深色模式 / 浅色模式切换
- 刷新页面后保留当前主题状态

### 输入区交互
- 输入框固定在页面底部
- 仅在有输入内容时显示发送按钮
- 支持图片上传与本地预览
- AI 响应过程中可点击“停止生成”
- 支持一键清空全部对话并恢复首页状态

### 其他细节
- 代码块支持语法高亮
- 代码块右上角支持复制按钮
- 支持快捷键操作：
  - `Enter` 发送消息
  - `Shift + Enter` 换行
- API Key 使用本地存储保存，不提交到仓库
- 本地存储字段名固定为：`LINGXI_API_KEY`

## API Key 配置说明

项目使用阿里云百炼提供的大模型服务。出于安全考虑，**不要将 API Key 提交到 GitHub 仓库**。

请在浏览器本地存储中手动设置：

```javascript
localStorage.setItem('LINGXI_API_KEY', '你的阿里云百炼API_KEY')
```
也可以在项目中设计首次输入 API Key 的交互弹窗，并将其保存到本地。
